# last.fm faves dumper

A python script to dump your last.fm faves list into simplified JSON:

``` json
[
    {
        "date": "2012/05/20", 
        "name": "Halcyon + On + On", 
        "artist": "Orbital"
    }, 
    {
        "date": "2012/05/20", 
        "name": "Iron Man", 
        "artist": "Black Sabbath"
    }, 
    {
        "date": "2012/05/20", 
        "name": "Summer of '69", 
        "artist": "Bryan Adams"
    }
]
```

Or YAML:

``` yaml
- artist: The Killers
  date: 2014/04/08
  name: I Can't Stay
- artist: Michael Penn
  date: 2014/04/03
  name: Walter Reed
- artist: The Handsome Family
  date: 2014/04/02
  name: Far From Any Road
```
